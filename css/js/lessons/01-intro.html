שיעור 1: מבוא ומשתנים

חזרה לעמוד הראשי

מה זה JavaScript?

JavaScript היא שפת תכנות שרצה בדפדפן ומאפשרת ליצור אתרים אינטראקטיביים.

משתנים

כדי לשמור נתונים בזיכרון, נשתמש במילה השמורה let או const.

משימה לתרגול:

1. צרו קובץ HTML בסיסי.
2. הוסיפו תגית <script>.
3. הגדירו משתנה בשם myName והכניסו לתוכו את השם שלכם.
4. הדפיסו את המשתנה לקונסול באמצעות console.log(myName);.
